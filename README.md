## Возможности

- **90 дней хранения** — письма хранятся 90 дней с автоматическим продлением при активности
- **Автоматическое переключение API** — при недоступности одного провайдера система автоматически переключается на резервный
- **Полноценный почтовый клиент** — получение, отправка и организация писем
- **Умные метки** — создавайте собственные метки для организации писем
- **Фильтрация спама** — автоматическое определение нежелательных писем
- **Адаптивный дизайн** — удобная работа на любых устройствах
- **Тёмная и светлая темы** — выбирайте комфортный режим отображения
- **Обязательное ознакомление с условиями** — пользователь должен прочитать условия использования

## Технологии

- **Next.js 16** — React фреймворк с App Router
- **TypeScript** — строгая типизация
- **Tailwind CSS 4** — стилизация
- **Zustand** — управление состоянием
- **shadcn/ui** — UI компоненты
- **Mail.tm API** — основной провайдер почты
- **1secmail API** — резервный провайдер

## Установка

### Требования

- Node.js 18.17 или выше
- pnpm (рекомендуется) / npm / yarn

### Локальная установка

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/rusmail.git
cd rusmail

# Установка зависимостей
pnpm install

# Запуск в режиме разработки
pnpm dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена

```bash
# Сборка приложения
pnpm build

# Запуск собранного приложения
pnpm start
```

## Конфигурация

### Переменные окружения

Приложение работает без дополнительной конфигурации, используя публичные API временной почты.

При необходимости можно настроить:

```env
# .env.local (опционально)
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

## API Провайдеры

RusMail использует следующие API для работы с временной почтой:

### Основной провайдер: Mail.tm
- Полнофункциональный API
- Поддержка отправки писем
- Надёжная работа

### Резервный провайдер: 1secmail
- Автоматическое переключение при недоступности основного
- Поддержка получения писем

### Механизм fallback

```typescript
// Автоматическое переключение между API
const APIs = [
  { name: 'mail.tm', priority: 1 },
  { name: '1secmail', priority: 2 },
]

// При ошибке система автоматически пробует следующий API
async function findAvailableAPI() {
  for (const api of APIs) {
    if (await checkAPIAvailability(api)) {
      return api
    }
  }
  return null
}
```

## Использование

### Получение почтового ящика

1. Перейдите на `/app`
2. Прочитайте и примите условия использования (обязательная прокрутка до конца)
3. Система автоматически создаст уникальный email адрес
4. Используйте адрес для регистрации на сайтах

### Организация писем

- **Метки** — создавайте собственные метки с цветовой маркировкой
- **Звёздочки** — отмечайте важные письма
- **Спам** — перемещайте нежелательные письма в спам
- **Поиск** — быстрый поиск по теме, отправителю и содержимому

### Срок хранения

- Письма хранятся **90 дней** с момента последней активности
- При каждом входе в почту срок автоматически продлевается
- После 90 дней неактивности данные удаляются безвозвратно
